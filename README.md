# random
> 无聊时做的一个抽奖机

### 关于 About

本网站由 [[0And1Story]](https://github.com/0And1Story) 开发

#### 使用方法：

1. 在文本框内输入要添加的条目，支持批量添加（使用空格/换行/逗号/顿号/etc.分割条目）
2. 点击`添加`（手机端为`+`）或点击回车键即可添加/批量添加条目
3. 若要删除条目，可点击表格右侧的`×`
4. 若要删除所有条目，可点击添加按钮旁的`清除所有`（手机端为`垃圾桶图标`）
5. 点击`开始`即可生成随机条目
